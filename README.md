# yangcong-dl
洋葱学院视频下载(无视VIP)
# 关于开源
本项目于2020年10月17日开源，采用MIT许可证，请自觉遵守。
## 相关开源库

# 使用方法
`python main.py`
或者[直接下载二进制版](https://github.com/ravizhan/yangcong-dl/releases)(不放心的可以自己编译)

# 原理
这得从洋葱学院网站上的一个越权漏洞(或者说是逻辑漏洞，我也不太确定...偶然发现的)说起
一句话概括:这个漏洞可以让普通用户，即非vip用户，下载到vip视频
